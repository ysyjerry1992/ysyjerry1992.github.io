为github博客绑定域名

今天在万网购买了域名，并将其绑定到了自己的博客上。在这里记录一下具体的方法。

域名注册

域名可以在很多地方购买，我这里是在万网上买的。

域名绑定

1.添加CNAME文件

在github仓库根目录下创建一个名为CNAME的文件，里面内容就是你的域名。

如example.com，不需要加http，www这样的前缀。

2.在万网添加解析

首先通过ping 你在github上的主页（×××.github.io）来获取ip地址。

然后在万网的域名解析界面按如下设置，记录值填刚刚记录下的ip地址。

这里有两种域名绑定的方式：A类型和CNAME类型，我采用的是A类型。

ps:ip ping一下域名就有了，也就没有涂。

3.验证

在命令行输入

dig yusiyuan.top +nostats +nocomments +nocmd

看到结果

;yusiyuan.top. IN A

yusiyuan.top. 5 IN A 103.245.222.133

4.等待

配置完了之后经过一段时间的等待就能通过域名访问在github上的博客啦。
