Posted on 11月 5 2014   |   In leetCode   |  

第十五题，找出所给字符串的最长公共前缀。

Write a function to find the longest common prefix string amongst an array of strings.

方法

我采用的方法，是先找出前两个字符串的公共前缀s，然后求s和下一个字符串的公共前缀。
